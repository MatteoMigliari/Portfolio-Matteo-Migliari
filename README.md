# Professional Portfolio | Matteo Migliari 

Welcome to my personal portfolio repository! This project showcases my journey as a Graphic and Digital Designer, blending creative vision with front-end development skills.

##  Live Demo
Check out the live site here: [https://github.com/MatteoMigliari/Portfolio-Matteo-Migliari]

##  Project Overview
This multi-page portfolio was designed with a focus on aesthetics, performance, and user experience, it serves as a digital gallery for my academic and professional case studies.

### Key Features
- **Modern UI/UX:** Clean design utilizing Glassmorphism and smooth CSS transitions.
- **Fully Responsive:** Seamlessly optimized for all devices (Mobile, Tablet, Desktop).
- **Sticky Navigation:** Persistent mobile safe-bar for enhanced accessibility.
- **Interactive Contact Form:** Fully functional form integrated with **EmailJS** for real-time messaging.
- **Project Showcases:** Custom lightboxes and Bootstrap carousels for high-fidelity project viewing.

## 🛠️ Tech Stack
This project was built using the following technologies and tools:

* **HTML5** for semantic web structure.
* **CSS3 / SASS (SCSS)** for advanced styling, variables, and clean code nesting.
* **Bootstrap 5** for grid management and responsive layout components.
* **JavaScript (ES6)** for UI logic, mobile menu transitions, and lightbox functionality.
* **Typed.js** for engaging header text animations.
* **EmailJS** for direct client-side email delivery integration.

## 📁 Repository Structure
- `index.html`: Main landing page and project grid.
- `about-me.html`: Digital CV and professional biography.
- `Contact.html`: Contact page with functional form.
- `/pagine_progetti`: Directory containing detailed case study pages.
- `/contenuti`: High-resolution mockups, project images, and assets.
- `/elementi`: Site-wide assets, icons, and branding elements.
- `style.css`: The main compiled CSS file.
